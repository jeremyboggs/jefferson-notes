# Notes on the State of Virginia
[![Build Status](https://travis-ci.org/waynegraham/jefferson-notes.png?branch=master)](https://travis-ci.org/waynegraham/jefferson-notes)


## Dependencies

* Rails
* bower (for managing JavaScript dependencies)

### Bower

```shell
$ npm install -g bower
```

## Running

```shell
$ bundle install
$ bower install
$ rake db:migrate
$ foreman start
```
